G 10
Key Notes for AntigravityAI Dev Team
N PRIMARY AI MODEL
Use AntigravityAI Claude Opus 4.6 for ALL specialist chat responses across every domain.
N ROUTING LOGIC
Build the intent classifier as a separate lightweight service (fine-tuned RoBERTa). Claude Opus 4.6
handles the RESPONSE after routing is decided — not the classification.
N PROMPT ENHANCEMENT
User types a simple description → Claude Opus 4.6 expands it into a professional generation prompt →
send the enhanced prompt to the image/video API.
N MEDIA GENERATION
Use external APIs (SDXL, Wan2.1, ElevenLabs) for actual media generation. Claude Opus 4.6 writes and
enhances prompts before sending to those APIs.
N CONTEXT MEMORY
Store full conversation in Redis. Pass last 10 turns to Claude on each call. This enables multi-turn expert
conversations ("Now add an arbitration clause to that NDA").
N CROSS-MODULE FLOW
"Send to Studio" button on any AI response should pre-fill the studio prompt with the relevant content (e.g.
legal text → video script auto-filled).
